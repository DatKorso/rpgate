## Компоненты интерфейса

### 1. Header
- Иконка приложения (Scroll)
- Название "RPGate"
- Подзаголовок

### 2. Chat Area (левая колонка)
- **Header**: "Приключение"
- **Messages**: ScrollArea с автоскроллом
  - Player messages (справа, синие)
  - GM messages (слева, серые)
  - System messages (центр, badge)
- **Loading Indicator**: прогресс обработки
- **Input**: поле ввода + кнопка отправки

### 3. Character Panel (правая колонка)
- **Header**: "Персонаж" с иконкой User
- **Inputs**: класс и биография
- **Actions**: сохранить, сбросить

### 4. Message Types

#### Player Message
```
┌────────────────────────────┐
│                      👤    │
│  ┌──────────────────────┐  │
│  │ Я осматриваюсь вокруг│  │
│  └──────────────────────┘  │
└────────────────────────────┘
```

#### GM Message
```
┌────────────────────────────┐
│  🤖                         │
│  ┌──────────────────────┐  │
│  │ Ты замечаешь скрытую │  │
│  │ дверь в стене...     │  │
│  └──────────────────────┘  │
└────────────────────────────┘
```

#### System Message
```
┌────────────────────────────┐
│     🎲 Бросок: d20=15      │
│     ✓ успех (+2)           │
└────────────────────────────┘
```

### 5. Loading Indicator

```
┌────────────────────────────────────────┐
│  ✓ Анализ правил ─ ⏳ Бросок ─ ○ Нарратив │
└────────────────────────────────────────┘
```

Этапы:
- ○ = не начат (пустой круг)
- ⏳ = в процессе (spinner)
- ✓ = завершён (галочка)

## Цветовая схема

### Primary Colors
- **Primary**: `#3B82F6` (синий)
- **Background**: `#FFFFFF` (белый)
- **Foreground**: `#0F172A` (тёмно-синий)

### Message Colors
- **Player**: `bg-primary text-primary-foreground`
- **GM**: `bg-muted text-foreground`
- **System**: `bg-muted text-muted-foreground`

### Status Colors
- **Success**: зелёный (`#22C55E`)
- **Fail**: красный (`#EF4444`)
- **Critical Success**: зелёный с border
- **Critical Fail**: красный с border

## Адаптивные breakpoints

- **Mobile**: < 640px (одна колонка)
- **Tablet**: 640px - 1023px (одна колонка)
- **Desktop**: 1024px+ (две колонки)

## Анимации

### Message появление
```css
animate-in fade-in-0 slide-in-from-bottom-2
```

### Loading spinner
```css
animate-spin
```

### Hover эффекты
```css
transition-colors hover:bg-primary/90
```

## Иконки (Lucide React)

- `Scroll` — логотип приложения
- `User` — игрок
- `Bot` — GM
- `Dices` — система/броски
- `Send` — отправить сообщение
- `Loader2` — загрузка
- `Save` — сохранить
- `Trash2` — удалить
- `TrendingUp/Down` — модификаторы
- `Check` — успех

## Accessibility

- Все интерактивные элементы доступны с клавиатуры
- Focus-visible стили для навигации
- ARIA labels через Radix UI
- Семантические HTML теги
- Screen reader friendly
