# Darwin/macOS Utilities

Filesystem
- `ls -la` тАФ list files
- `cd path` тАФ change directory
- `find . -name "pattern"` тАФ search files
- `rg "regex"` тАФ fast text search (install ripgrep)

Editing/Viewing
- `sed -n '1,200p' file` тАФ print file lines
- `bat file` тАФ pretty cat (optional)

Clipboard/Apps
- `pbcopy` / `pbpaste` тАФ clipboard integration
- `open file_or_url` тАФ open with default app

Git
- `git status` / `git add -p` / `git commit -m "msg"` / `git log --oneline --graph --decorate`
